# RhizomeAI Wikipedia Prototype

A Wikipedia-powered rhizomatic interface:
- Search for an article
- Node uses the article title and main image (if available)
- Click/select nodes to see article hyperlinks in the upper-left link panel
- Click hyperlinks to add connected article nodes
- Node selection syncs across graph, chat, and link panel

## Run locally

1. Install dependencies

```bash
npm install
```

2. Optional env file

```bash
cp .env.example .env
```

3. Start

```bash
npm start
```

4. Open

`http://localhost:8787`

## API

- `GET /api/health`
- `GET /api/article?q=<query>`

## Make it live from GitHub

GitHub Pages alone will not run this app (it needs Node API endpoints).
Use a Node host like Render.

### Render steps

1. Push project to GitHub.
2. In Render: **New +** -> **Web Service**.
3. Connect your repo.
4. Build command: `npm install`
5. Start command: `npm start`
6. Deploy.

Your live URL will be generated by Render.
